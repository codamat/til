# JIS Q 27000

企業が情報漏洩などのリスクを防ぎ、安全に情報を管理するための仕組み（ISMS：情報セキュリティマネジメントシステム）に関する「言葉の定義」や「概要」をまとめた日本産業規格。
国際規格ISO/IEC 27000の翻訳版であり、情報セキュリティの基本となる「3要素（機密性・完全性・可用性）」や専門用語を共通認識として定義。 

JIS Q 27000は単体ではなく、関連する規格群（ファミリー）の一部。 

- **JIS Q 27000**: ==用語と定義==（辞書の役割）。
- **JIS Q 27001**: ISMS認証取得のための「==要求事項==」（ルールそのもの）。
- **JIS Q 27002**: 具体的なセキュリティ対策の「ガイドライン」（==実践方法==）。 

JIS Q 27001/27002をベースとした、特定の分野に特化した規格。

- **JIS Q 27014（情報セキュリティガバナンス）**: 
  組織の経営陣が情報セキュリティをどう指揮・管理すべきかという==ガバナンス==に焦点を当てた規格。
- **JIS Q 27017（クラウドセキュリティ）**: 
  クラウドサービスの提供・利用に関する情報セキュリティ管理策。==クラウド特有のリスク==（データの所在など）が対象。
- **JIS Q 27018（クラウド個人情報保護）**: 
  パブリッククラウド上でPII（Personally Identifiable Information：==個人識別情報==）を扱う際の保護に特化した規格。
